About PII Detection

This portal detects personally identifiable information (PII) such as Aadhaar numbers, PAN, phone numbers, emails and more. It can de-identify (mask) detected values so data can be used safely for analysis while preserving privacy.

Accurate Masking

Regex + heuristic based detection with configurable masks to suit formats (India-specific patterns included).

Local Processing

Process text locally or upload CSVs. No data is sent to third-party services by default.

Report Generation

Generate reports summarizing detected PII counts and masked outputs for auditing.
